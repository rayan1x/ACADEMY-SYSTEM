Modalités de paiement
Carte Bancaire CIH
Seuls les paiements par carte bancaire CIH sont acceptés.
Les transactions sont sécurisées par la CIH Bank.
Les commandes doivent être payées depuis votre espace client.
Il n'est pas obligatoire d'envoyer le reçu de paiement.
Délai de validation : Activation instantanée si la transaction est réussie.
PayPal
Le paiement par PayPal est réservé aux clients hors Maroc.
Les transactions sont sécurisées par PayPal.
Les commandes doivent être payées depuis votre espace client.
Il n'est pas obligatoire d'envoyer le reçu de paiement.
Délai de validation : Activation instantanée si la transaction est réussie.
Le paiement par PayPal est réservé aux clients hors Maroc.
